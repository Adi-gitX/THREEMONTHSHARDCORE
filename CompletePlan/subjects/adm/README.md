## üí° Proof Writing Tips

### Structure
```
**Theorem:** [Statement to prove]

**Proof:**
[Method: Direct/Contradiction/Contrapositive/Induction]

1. [Assumptions/Given]
2. [Logical steps with justification]
3. [Each step follows from previous]
...
n. [Conclusion] ‚àé

**Therefore,** [restate theorem]
```

### Common Mistakes to Avoid
- ‚ùå Assuming what you're trying to prove
- ‚ùå Using examples as proof
- ‚ùå Missing edge cases
- ‚ùå Circular reasoning
- ‚ùå Unclear logic jumps

### Good Practices
- ‚úÖ State method explicitly
- ‚úÖ Define all variables
- ‚úÖ Justify each step
- ‚úÖ Use proper notation
- ‚úÖ End with clear conclusion
