# CompletePlan Directory

This directory contains the complete semester plan and schedule.

## Contents

### Semester6.csv
A comprehensive 90-day (plus bonus day 91) study plan covering:
- Operating Systems (OS)
- DevOps
- Deep Learning (DL) / Applied Machine Learning (AML)
- Advanced Discrete Mathematics (ADM)
- Data Structures & Algorithms (DSA)
- Interview Preparation
- Project Tasks
- Networking

## File Format

The CSV file contains the following columns:
- Title (Day number and date)
- Date
- Week
- Phase
- Contest/Event
- Subject allocations (OS, DevOps, DL/AML, ADM, DSA, Interview Prep, Project, Networking)
- Action items for each subject
- Completion tracking
- Energy levels
- Key learnings
- Blockers
- Tomorrow's priorities

## Note on Day 91

The plan includes a bonus "Day 91" which serves as a celebration and wrap-up day after completing the core 90-day journey. This is intentional to mark the completion and transition after the intensive 90-day period.

## Usage

This CSV can be:
- Imported into spreadsheet applications
- Used for progress tracking
- Referenced for daily planning
- Analyzed for time allocation patterns
