This directory contains templates for quick reference when creating daily documentation.

## Available Templates

### 1. `day-readme-template.md`
Main daily README template with:
- Date and status
- Today's focus
- Tasks completed
- Achievements
- Summary
- Reflection
- Energy, productivity, and mood tracking
- Navigation links

### 2. `learnings-template.md`
Daily learnings template with:
- Key takeaways
- Concepts studied
- Skills practiced
- Problems solved
- Resources used
- Notes section

### 3. `tasks-template.md`
Task tracking template with:
- High priority tasks
- Medium priority tasks
- Low priority / Ideas
- Completed tasks
- Progress tracking

### 4. `notes-template.md`
Daily notes template with:
- General notes
- Ideas and thoughts
- Important reminders
- Code snippets
- Links and references
- Questions

### 5. `weekly-summary-template.md`
Weekly summary template with:
- Week overview and focus
- Accomplishments and tasks
- Learning highlights
- Challenges and solutions
- Statistics and metrics
- Reflection and next week goals

### 6. `monthly-summary-template.md`
Monthly summary template with:
- Month overview and theme
- Major accomplishments
- Learning and growth metrics
- Detailed statistics
- Challenges and resilience
- Deep reflection
- Next month goals

## Using Templates

### Manual Copy
Simply copy the content from any template file and paste it into your daily file.

### Using the Script
Use the `generate-template.py` script to automatically create properly formatted daily files:

```bash
python3 scripts/generate-template.py <day_number>
```

See [scripts/README.md](../scripts/README.md) for more details.

## Customization

Feel free to customize these templates to match your personal workflow and preferences. The key is consistency!
